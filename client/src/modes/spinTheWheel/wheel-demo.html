About This Implementation

This Canvas-based spinning wheel component features:

  • Three Readability Tiers:
    • 0-24 entries: All labels visible on wheel
    • 25-80 entries: Windowed labels (only near pointer)
    • 81+ entries: Outer label ring with leader lines
  • Performance Optimizations:
    • Offscreen canvas caching for segments
    • 60 FPS animation with requestAnimationFrame
    • Efficient text rendering with ellipsis
  • Advanced Features:
    • Weighted entries support
    • Deterministic randomness with seed
    • Smooth easing animations
    • Responsive design
    • Accessibility (ARIA announcements)
